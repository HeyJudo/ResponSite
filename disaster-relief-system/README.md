# Disaster Relief System

A Java-based monorepo for the disaster relief system application.

## Project Structure

This monorepo contains the following modules:

- **client-frontend/**: Client-side frontend application
- **server-backend/**: Server-side backend application
- **docs/**: Project documentation
- **lib/**: Shared libraries and utilities

## Overview

The Disaster Relief System is designed to help coordinate and manage disaster relief efforts efficiently.

## Prerequisites

- Java Development Kit (JDK) 11 or higher
- Maven or Gradle
- IntelliJ IDEA (recommended IDE)
